# 🚀 Python & AI 开发学习路径

## 📋 课程大纲

### Phase 1: Python 基础 (Day 1-7)
- ✅ **Day 1**: 对话机器人 - 类、函数、输入输出
- 📅 **Day 2**: 数据管理 - 待办事项列表 (列表、字典、文件操作)
- 📅 **Day 3**: API调用 - 天气查询工具 (requests、JSON、API)
- 📅 **Day 4**: 数据分析 - CSV处理与可视化 (pandas、matplotlib)
- 📅 **Day 5**: 错误处理 & 日志 - 健壮的爬虫程序
- 📅 **Day 6**: OOP项目 - 简易图书管理系统
- 📅 **Day 7**: 综合项目 - 个人财务管理工具

### Phase 2: LangChain & 大模型 (Day 8-14)
- 📅 **Day 8**: 连接OpenAI API - 第一个AI应用
- 📅 **Day 9**: Prompt工程 - 提示词设计与优化
- 📅 **Day 10**: LangChain基础 - Chains & Prompts
- 📅 **Day 11**: RAG系统 - 文档问答机器人
- 📅 **Day 12**: 记忆系统 - 有状态的对话
- 📅 **Day 13**: 工具调用 - 让AI使用外部工具
- 📅 **Day 14**: 综合项目 - 智能客服系统

### Phase 3: AI智能体 (Day 15-21)
- 📅 **Day 15**: Agent基础 - ReAct模式
- 📅 **Day 16**: 多Agent协作 - 团队智能体
- 📅 **Day 17**: 网页爬虫Agent - 自主信息收集
- 📅 **Day 18**: 代码生成Agent - AI编程助手
- 📅 **Day 19**: 数据库Agent - 自然语言查询
- 📅 **Day 20**: API自动化 - 工作流编排
- 📅 **Day 21**: 综合项目 - 个人AI助手

### Phase 4: 实战项目 (Day 22-30)
- 📅 **Day 22-24**: 企业级项目 - 智能知识库
- 📅 **Day 25-27**: 创业项目 - AI内容创作平台
- 📅 **Day 28-30**: 求职作品集整理 & GitHub优化

## 🎯 每天学习流程 (建议1-2小时)

### 1️⃣ 动手编码 (30分钟)
- 运行当天的Python脚本
- 修改代码，尝试不同参数
- 完成小挑战

### 2️⃣ 理解原理 (20分钟)
- 阅读代码注释
- 理解每个函数的作用
- 思考如何改进

### 3️⃣ 扩展练习 (30分钟)
- 添加新功能
- 修复bug
- 优化用户体验

### 4️⃣ 记录成果 (10分钟)
- 提交到GitHub
- 写简短的学习笔记
- 截图运行结果

## 💡 学习建议

### 保持动力
1. **每天进步一点点** - 即使只有30分钟也好
2. **记录成长** - 截图、写笔记、发朋友圈
3. **找人一起学** - 加入学习社群
4. **教别人** - 教学相长

### 遇到困难时
- 🤔 先自己思考5分钟
- 🔍 搜索错误信息
- 📖 查阅文档
- 💬 请教学习伙伴

### 建立习惯
- 固定学习时间（建议早上或晚上）
- 创建专属学习空间
- 关闭手机通知，专注编码
- 每完成一个阶段，给自己小奖励
